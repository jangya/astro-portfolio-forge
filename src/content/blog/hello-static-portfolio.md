This portfolio is intentionally static. Markdown powers the writing, Astro renders the pages at build time, and Decap CMS gives non-developers a friendly editing layer without adding a server.

The goal is a site that feels precise, personal, and easy to fork. Change the configuration, write a few posts, choose featured GitHub repositories, and publish to GitHub Pages.
